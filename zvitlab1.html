Висновок

У лабораторній роботі створено першу версію HTML-сторінки з використанням семантичної структури та основних елементів: заголовків, зображень, таблиці, списків, форми та гіперпосилання. Обраною предметною галуззю став каталог культурно-історичних та туристичних об’єктів України, що має інформативну й візуально зручну подачу даних для користувача.

КАСКАДНІ ТАБЛИЦІ СТИЛІВ. СЕЛЕКТОРИ. ІДЕНТИФІКАТОРИ. СТИЛЬОВЕ ОФОРМЛЕННЯ ТЕКСТОВИХ ЕЛЕМЕНТІВ В HTML-ДОКУМЕНТАХ.

1. Селектори тегів

Селектори тегів застосовуються напряму до HTML-елементів. Наприклад, стиль для всіх параграфів:

p {
  line-height: 1.6;
}

HTML-приклад:

<p>Це приклад параграфа</p>

2. Селектори класів

Селектори класів використовуються для стилізації груп елементів з однаковим класом:

.subtitle {
  font-style: italic;
  color: #3a3a3a;
}

HTML-приклад:

<h2 class="subtitle">Визначні місця</h2>

3. Селектори ідентифікаторів

Селектори ID застосовуються до унікальних елементів:

#main-title {
  text-transform: uppercase;
  border-bottom: 2px solid #2a4d8f;
}

HTML-приклад:

<h1 id="main-title">Каталог об’єктів</h1>

4. Інші селектори

Приклади комбінаційних та атрибутних селекторів:

h3 + ul {
  padding-left: 25px;
}

form > input, form > textarea {
  border: 1px solid #aaa;
  padding: 5px;
}

input[type="email"] {
  background-color: #eef7ff;
}

* {
  box-sizing: border-box;
}

HTML-приклад (форма):

<form>
  <input type="text" placeholder="Ім’я">
  <input type="email" placeholder="Email">
  <textarea placeholder="Повідомлення"></textarea>
</form>

Також застосовано стилі до шрифтів, тексту, таблиць, форм, списків і зображень, включаючи просунуті можливості CSS (анімації, псевдокласи). Тема: ВЕРСТКА HTML-ДОКУМЕНТУ. БЛОКОВА ВЕРСТКА. ВЕРСТКА ЗАСОБАМИ CSS та FLEXBOX

Завдання №2.1: Таблична верстка

Було створено веб-сторінку з табличною версткою, яка містить заголовок, бічне меню, основний вміст та додаткову інформацію.

    </td>
  </tr>
  ...
</table>

Фрагмент CSS-коду:

.layout {
  width: 95%;
  margin: 20px auto;
  border-collapse: collapse;
  background-color: #ffffff;
  ...
}

Висновки

У ході виконання лабораторної роботи було засвоєно основні принципи табличної та блокової верстки, а також практичне використання сучасної технології Flexbox. Реалізовано різні типи верстки для однієї й тієї ж теми, що дозволяє порівняти підходи та обрати оптимальний для подальшого використання.

Тема: АДАПТИВНА ВЕРСТКА HTML-ДОКУМЕНТУ

Метою лабораторної роботи було створення вебсторінки з використанням CSS Grid-сітки та медіа-запитів для реалізації адаптивної верстки на трьох рівнях: десктоп, планшет і мобільний пристрій.

Десктопна версія

Для екранів шириною понад 992px використовується трьохколонкова структура з використанням CSS Grid.

Фрагмент HTML-коду:

<div class="container">
  <header>...</header>
  <div class="sidebar">...</div>
  <main>...</main>
  <aside class="extra">...</aside>
  <footer>...</footer>
</div>

Фрагмент CSS-коду:

.container {
  display: grid;
  grid-template-columns: 20% 55% 25%;
  grid-template-areas:
    "header header header"
    "sidebar main extra"
    "footer footer footer";
  min-height: 100vh; /* ← зміна тут */
}

Планшетна версія

При ширині екрана менше 992px макет перебудовується у дві колонки: бічне меню переміщується зверху.

Фрагмент CSS-коду:

@media (min-width: 600px) and (max-width: 1023px) {
  .container {
    flex-direction: row;
    flex-wrap: wrap;
  }

  .sidebar {
    width: 25%;
    order: 1;
    background-color: #d9e4f5;
  }

  .main {
    width: 75%;
    order: 2;
  }

  .extra {
    width: 100%;
    order: 3;
    background-color: #eef3fa;
  }

  .footer {
    width: 100%;
    order: 4;
  }
}

Мобільна версія

Для екранів шириною менше 768px усі блоки виводяться один під одним у вигляді одноколонкової сітки.

Фрагмент CSS-коду:

.container {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
  max-width: 1200px;
  margin: 0 auto;
  background-color: #fff;
  box-shadow: 0 0 10px rgba(0,0,0,0.1);
}
  .sidebar,
.main,
.extra {
  width: 100%;
}

Висновки

У ході виконання лабораторної роботи було створено адаптивний вебсайт з використанням CSS Grid-сітки. Застосування медіа-запитів дозволило реалізувати гнучке відображення контенту на пристроях різної ширини, забезпечивши зручність перегляду на ПК, планшетах і смартфонах. Макет успішно адаптується до трьох основних розмірів екранів.
